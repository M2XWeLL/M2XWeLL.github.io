Модуль: Модерация

mute — замьютить участника в данном текстовом канале(работает по ID)

Пример: ".mute |ID|"

unmute — размьютить участника в данном текстовом канале(работает по ID)

Пример: ".unmute |ID|"

kick — выгнать участника с сервера(Работает по ID)

Пример: ".kick |ID| |reason|"

ban — заюанить участника на сервере(работает по ID)

Пример: ".ban |ID| |дни удвления сообщений(от 0 до 7)| |reason|"

server — Информация о сервере

Пример: ".server"
